# SCT_WD_1 - Interactive Landing Page

**SkillCraft Technology Web Development Task 1**

A responsive landing page with an interactive fixed navigation bar that changes style on scroll and hover.

## 🚀 Live Demo
[https://mounikainturi.github.io/SCT_WD_1/](https://mounikainturi.github.io/SCT_WD_1/)

## 📋 Task Requirements Met

| Requirement | Implementation |
| --- | --- |
| Fixed navigation menu visible on all pages | `position: fixed` navbar stays on screen while scrolling |
| Menu changes color/style when scrolled | Transparent → solid white with blur + shadow after 50px scroll |
| Menu changes when hovering over item | Animated underline + color shift to `#f6ad55` on hover |
| Responsive design | Hamburger menu for mobile, CSS Grid for services |

## ✨ Features

1. **Interactive Navbar**: Changes from transparent to glassmorphism style on scroll
2. **Hover Effects**: Smooth underline animation on menu items
3. **Mobile Friendly**: Hamburger menu + responsive grid layout
4. **Smooth Scrolling**: Click nav links to jump to sections
5. **Human-Friendly UI**: Warm colors, clear copy, micro-interactions

## 🛠️ Tech Stack

- **HTML5**: Semantic structure
- **CSS3**: Flexbox, Grid, backdrop-filter, animations
- **Vanilla JS**: Scroll listener, mobile menu toggle
- **No frameworks**: Pure HTML/CSS/JS as required
